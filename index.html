About The LED Cube

The cube has 512 blue LEDs which make up it's 8 layers and 32 columns. All the horizontal layers are the positive pins of the LEDs soldered together. All the negative pins of the LEDs are treated as columns of this matrix cube. These are all wired to an Arduino.

An Arduino is a single-board micro-controller, intended to make the application of interactive objects or environments more accessible. Arduino is an open-source platform for building electronic projects. The Arduino platform consists of a physical programmable microcontroller (circuit board) and an integrated development environment to write and upload code to the physical board. The Arduino IDE uses a simplified version of C++.
